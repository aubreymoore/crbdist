The objective of this project is to build a public database to facilitate sharing information on new geographical occurrence records for the coconut rhinoceros beetle (CRB), Oryctes rhinoceros, a major pest of coconut and oil palm. CRB is currently spreading among Pacific Islands.

Instead of building,maintaining, and hosting a custom database, I decided to use the Global Biodiversity Information Facility as the project database which already contains occurrence records for CRB. The first step is to build a Darwin Core Archive (DwCA) which is a biodiversity informatics data used for uploading data to GBIF.
