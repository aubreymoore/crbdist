The objective of this project is to build a public database to facilitate sharing information on new geographical occurrence records for the coconut rhinoceros beetle (CRB), Oryctes rhinoceros, a major pest of coconut and oil palm which is currently spreading among Pacific Islands.

Instead of building, maintaining, and hosting a custom database, I decided to publish data to the Global Biodiversity Information Facility (GBIF) which already contains occurrence records for CRB. My goal is to fill data gaps in the GBIF data for CRB and to provide code to facilitate easy access to all GBIF occurrence records for CRB. The first step is to build a Darwin Core Archive (DwCA) which is a biodiversity informatics data used for uploading data to GBIF. The archive will contained first occurrence records of CRB for islands in the Pacific and Indian Oceans and elsewhere.

Darwin Core Archive (under construction)

The Darwin Core Archive DwCA.zip is a compressed ZIP file containing the following text files.

DwCA/occurrence.txt A delimited text file containing first country and island occurrence records for CRB.

DwCA/meta.xml A XML text file containing definitions for fields in occurrence.txt.

DwCA/eml.xml A XML text file containing metadata describing the DwCA dataset.
